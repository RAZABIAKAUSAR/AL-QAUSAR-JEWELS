# AL-QAUSAR-JEWELS

**Project Type:** Personal Full Stack Project (Beginner Level)  
**Website:** Online Jewelry E-commerce  

## Description
Al Qausar Jewels is an **online jewelry brand website** built from scratch as a learning project. The project demonstrates building an e-commerce workflow including:  

- Product listings and catalog display  
- Customer order collection and management  
- Basic order-to-invoice workflow automation using Google Workspace tools  
- Frontend design with HTML, CSS, and JavaScript  

## Tools & Technologies
- HTML, CSS, JavaScript  
- Google Forms, Sheets, Docs (for workflow automation)  
- GitHub (version control)  

## Status
Project is **in progress / beginner level**, focusing on learning full-stack development, automation, and web design concepts.
